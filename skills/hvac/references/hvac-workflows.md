# HVAC Workflows

## Installation Workflow

### State Flow
```
接单 → 上门勘测 → 报价确认 → 施工安装 → 调试验收 → 客户签字 → 完成
```

### Checklist

**勘测阶段**:
- [ ] 确认电源电压 (220V/380V)
- [ ] 确认电源线径是否足够
- [ ] 确认安装位置（室内机、室外机）
- [ ] 确认排水位置
- [ ] 确认墙体承重
- [ ] 测量管长距离

**安装阶段**:
- [ ] 安装室内机（水平、固定、留足检修空间）
- [ ] 安装室外机（通风良好、固定牢固）
- [ ] 连接铜管（喇叭口连接、抽真空 15 分钟以上）
- [ ] 保压测试（24 小时保压）
- [ ] 连接排水管（确认排水顺畅）
- [ ] 连接电源线（确认线径、接地）
- [ ] 充注冷媒（按铭牌标注量）
- [ ] 包扎保温棉

**调试阶段**:
- [ ] 制冷模式测试（运行 15 分钟，测量出风温度）
- [ ] 制热模式测试（运行 15 分钟，测量出风温度）
- [ ] 排水测试（确认无漏水）
- [ ] 噪音测试（室内机、室外机）
- [ ] 遥控器功能测试
- [ ] 客户验收签字

## Repair Workflow

### State Flow
```
接单 → 扫码识别 → 故障诊断 → [报价确认] → 维修处理 → 测试验收 → 收费 → 完成
                                    ↓
                              [需要配件] → 暂停 → 领料 → 继续
```

### Common Fault Categories

| Category | Symptoms | Common Causes |
|----------|----------|--------------|
| 不制冷 | 出风不冷、温度降不下来 | 缺冷媒、压缩机故障、电容损坏 |
| 不制热 | 出风不热、四通阀不换向 | 四通阀故障、电辅热损坏 |
| 不启动 | 不通电、不运转 | 电源问题、电路板故障、电容损坏 |
| 漏水 | 室内机滴水、排水管溢出 | 排水管堵塞、安装不水平 |
| 噪音大 | 异响、振动 | 风机故障、压缩机异常、安装松动 |
| 异味 | 霉味、烧焦味 | 蒸发器发霉、电路过热 |
| 遥控器失灵 | 无法控制 | 遥控器电池、接收头故障 |

## Maintenance Workflow

### State Flow
```
接单 → 扫码识别 → 按清单检查 → 清洗保养 → 更换配件 → 记录数据 → 完成
```

### Maintenance Checklist

**室内机**:
- [ ] 过滤网（拆下清洗，晾干后装回）
- [ ] 蒸发器（用专用清洗剂清洗）
- [ ] 排水管（注入清水确认排水顺畅）
- [ ] 风机（检查运转是否正常）
- [ ] 出风温度（制冷 ≤ 12°C，制热 ≥ 40°C）
- [ ] 遥控器（检查电池）

**室外机**:
- [ ] 冷凝器（用高压水枪清洗）
- [ ] 风机（检查运转是否正常）
- [ ] 运行电流（对比铭牌额定值）
- [ ] 制冷剂压力（低压 0.4-0.6 MPa，高压 1.5-2.0 MPa）
- [ ] 电气连接（检查有无松动、发热）
- [ ] 支架（检查是否牢固、有无锈蚀）

**系统**:
- [ ] 制冷效果（温差 ≥ 8°C）
- [ ] 制热效果（温差 ≥ 15°C）
- [ ] 噪音水平（对比正常值）
- [ ] 漏水检查
- [ ] 记录运行参数

### Maintenance Schedule

| Usage | Recommended Interval |
|-------|---------------------|
| 家用（普通） | 每年 1 次 |
| 家用（高频） | 每年 2 次 |
| 商用（办公室） | 每季度 1 次 |
| 商用（餐饮/工厂） | 每月 1 次 |
| 机房/数据中心 | 每月 1 次 |

## Inspection Workflow

### State Flow
```
接单 → 扫码签到 → 逐项检查 → 拍照记录 → 生成报告 → 完成
```

### Inspection Items

- [ ] 设备外观（有无损坏、锈蚀）
- [ ] 运行状态（是否正常运转）
- [ ] 电气安全（漏电保护、接地）
- [ ] 制冷/制热效果
- [ ] 噪音水平
- [ ] 排水系统
- [ ] 支架固定
- [ ] 冷媒泄漏检测
