# Solar Workflows

## Installation

### State Flow
```
接单 → 现场勘测 → 方案设计 → 报价确认 → 施工安装 → 并网验收 → 完成
```

### Installation Checklist
- [ ] 屋顶/场地条件评估
- [ ] 承重计算
- [ ] 遮挡分析
- [ ] 电气设计
- [ ] 安装支架
- [ ] 安装组件
- [ ] 连接线缆
- [ ] 安装逆变器
- [ ] 安装配电箱
- [ ] 接地施工
- [ ] 并网申请
- [ ] 电力局验收
- [ ] 并网发电

## Cleaning

### State Flow
```
接单 → 扫码签到 → 安全检查 → 清洗组件 → 检查连接 → 记录数据 → 完成
```

### Cleaning Frequency

| Environment | Frequency |
|------------|-----------|
| 城市 | 每年 2-4 次 |
| 工业区 | 每年 4-6 次 |
| 沙漠/灰尘多 | 每月 1 次 |
| 农业区 | 每年 3-4 次 |

## Inspection

### State Flow
```
接单 → 扫码签到 → 外观检查 → 电气测试 → 红外热成像 → 生成报告 → 完成
```

### Inspection Checklist
- [ ] 组件外观（破损、蜗牛纹、变色）
- [ ] 支架固定（锈蚀、松动）
- [ ] 线缆连接（老化、松动）
- [ ] 接线盒（进水、烧蚀）
- [ ] 逆变器运行状态
- [ ] 绝缘电阻测试
- [ ] 接地电阻测试
- [ ] 红外热成像（热斑检测）
- [ ] IV 曲线测试（可选）

## Repair

### Common Faults

| 故障 | 原因 | 处理 |
|------|------|------|
| 发电量下降 | 灰尘、组件衰减、遮挡 | 清洗、更换 |
| 逆变器报警 | 电网异常、组件故障 | 检查电网、更换 |
| 热斑 | 组件缺陷、鸟粪遮挡 | 清洗、更换组件 |
| 绝缘故障 | 线缆老化、接头进水 | 更换线缆 |
| 通信故障 | 监控模块故障 | 更换模块 |
