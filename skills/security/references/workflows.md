# Security Workflows

## Installation

### State Flow
```
接单 → 现场勘测 → 方案设计 → 报价确认 → 施工安装 → 调试验收 → 培训交付 → 完成
```

### Installation Checklist (Camera)
- [ ] 确认安装位置和角度
- [ ] 安装支架
- [ ] 安装摄像头
- [ ] 布线（网线/电源线）
- [ ] 连接POE交换机
- [ ] 配置IP地址
- [ ] 配置录像计划
- [ ] 调整镜头焦距
- [ ] 测试夜视效果
- [ ] 测试远程访问
- [ ] 客户验收

## Maintenance

### Camera Checklist
- [ ] 镜头清洁
- [ ] 遮挡检查
- [ ] 红外灯正常
- [ ] 图像质量检查
- [ ] 录像回放测试
- [ ] 网络连接正常
- [ ] 存储状态检查
- [ ] 固件版本检查
- [ ] 防水密封检查

### Access Control Checklist
- [ ] 读卡器功能正常
- [ ] 电锁动作正常
- [ ] 出门按钮正常
- [ ] 门磁状态正确
- [ ] 联动报警测试
- [ ] 电池备用电源

## Repair

### Common Faults

| 故障 | 原因 | 处理 |
|------|------|------|
| 图像模糊 | 镜头脏、失焦 | 清洁、调焦 |
| 夜视效果差 | IR灯衰减 | 更换IR灯 |
| 离线 | 网络/电源故障 | 检查网络和电源 |
| 录像丢失 | 硬盘故障 | 更换硬盘 |
| 门禁不开 | 读卡器/电锁故障 | 更换设备 |
| 误报警 | 灵敏度/环境干扰 | 调整灵敏度 |
