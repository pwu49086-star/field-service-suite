# Fire Safety Workflows

## Inspection (Primary)

### State Flow
```
接单 → 扫码签到 → 逐项检查 → 拍照记录 → 生成报告 → 完成
                                    ↓
                              [不合格项] → 生成整改工单
```

### Extinguisher Checklist
- [ ] 外观（无锈蚀、无损坏、标识清晰）
- [ ] 压力表（绿色区域）
- [ ] 保险销（完好）
- [ ] 喷管（无堵塞、无破损）
- [ ] 标签（生产日期、有效期清晰）
- [ ] 放置位置（固定、易取、标识清晰）
- [ ] 数量配置（符合规范）

### Alarm System Checklist
- [ ] 主机运行正常
- [ ] 各回路正常
- [ ] 探测器报警测试
- [ ] 手动报警按钮测试
- [ ] 声光报警测试
- [ ] 联动功能测试
- [ ] 备用电源测试

### Sprinkler Checklist
- [ ] 喷淋头无遮挡
- [ ] 管道无锈蚀
- [ ] 阀门开启状态
- [ ] 水压正常
- [ ] 末端试水测试

## Repair/Replacement

### Common Faults

| 故障 | 原因 | 处理 |
|------|------|------|
| 灭火器压力不足 | 药剂泄漏、温度影响 | 重新充装 |
| 探测器误报 | 灰尘、灵敏度过高 | 清洗、调整灵敏度 |
| 探测器不报 | 探测器损坏、线路故障 | 更换探测器 |
| 喷淋头漏水 | 密封老化、管道损坏 | 更换喷淋头 |
| 应急灯不亮 | 电池失效、电路故障 | 更换电池 |

## Fire Drill

### State Flow
```
计划制定 → 通知参与人员 → 实施演练 → 记录过程 → 总结评估 → 归档 → 完成
```

### Drill Record
- 演练日期时间
- 参与人数
- 演练类型（疏散/灭火/综合）
- 疏散用时
- 发现问题
- 改进措施
