# Elevator Workflows

## Maintenance (Primary)

### State Flow
```
接单 → 扫码签到 → 按清单检查 → 润滑保养 → 记录数据 → 客户签字 → 完成
```

### Maintenance Checklist

**机房**:
- [ ] 机房温度（5-40°C）
- [ ] 机房湿度（< 85%）
- [ ] 照明正常
- [ ] 通风正常
- [ ] 无杂物堆放
- [ ] 地面清洁

**曳引系统**:
- [ ] 曳引机油位正常
- [ ] 曳引机油质正常
- [ ] 曳引机振动/噪音正常
- [ ] 曳引轮槽磨损检查
- [ ] 钢丝绳断丝检查
- [ ] 钢丝绳张力均匀

**控制系统**:
- [ ] 控制柜指示灯正常
- [ ] 接触器无烧蚀
- [ ] 继电器动作正常
- [ ] 变频器无报警

**轿厢**:
- [ ] 照明正常
- [ ] 通风正常
- [ ] 按钮功能正常
- [ ] 显示正常
- [ ] 轿厢清洁
- [ ] 紧急通讯正常

**门系统**:
- [ ] 轿门开关顺畅
- [ ] 轿门间隙合格
- [ ] 安全触板/光幕正常
- [ ] 层门开关顺畅
- [ ] 层门锁紧正常

**安全部件**:
- [ ] 限速器动作灵活
- [ ] 安全钳状态正常
- [ ] 缓冲器状态正常
- [ ] 限位开关正常

**运行测试**:
- [ ] 平层精度（±5mm）
- [ ] 运行噪音（< 65dB）
- [ ] 运行振动正常
- [ ] 消防功能测试
- [ ] 紧急停止测试

## Annual Inspection

### State Flow
```
接单 → 准备资料 → 现场检验 → 整改项处理 → 取得报告 → 更新证书 → 完成
```

### Inspection Preparation

- [ ] 维保记录完整
- [ ] 维修改造记录
- [ ] 使用证书有效
- [ ] 注册登记证
- [ ] 安全管理制度
- [ ] 操作人员证书

## Repair

### Common Faults

| 故障 | 原因 | 处理 | 配件 |
|------|------|------|------|
| 不运行 | 安全回路断开、控制柜故障 | 检查安全回路 | 控制板 |
| 平层不准 | 编码器、导靴磨损 | 更换编码器 | 编码器、导靴 |
| 门故障 | 门机故障、门锁触点 | 更换门机 | 门机、门滑块 |
| 困人 | 安全回路动作、停电 | 盘车救援 | - |
| 振动大 | 导靴磨损、钢丝绳张力不均 | 更换导靴 | 导靴 |
| 噪音大 | 轴承磨损、齿轮异常 | 更换轴承 | 轴承 |
