# Appliance Repair Workflows

## Repair Workflow (Primary)

### State Flow
```
接单 → 扫码识别 → 故障诊断 → 报价确认 → 维修处理 → 测试验收 → 收费 → 完成
                                      ↓
                                [需要配件] → 暂停 → 领料 → 继续维修
```

### Diagnosis Process

1. **扫码识别** — 扫描设备条码/二维码，调取设备信息和维修历史
2. **客户描述** — 记录客户描述的故障现象
3. **现场检查** — 技师实际检查设备状态
4. **故障定位** — 确定故障原因和需要更换的配件
5. **报价** — 向客户报价（配件 + 工时费）
6. **客户确认** — 客户同意后开始维修

### Maintenance Checklist

**洗衣机**:
- [ ] 清洗内筒（专用清洗剂）
- [ ] 检查进水管（有无老化、漏水）
- [ ] 检查排水管（有无堵塞）
- [ ] 清洁过滤网
- [ ] 检查门封条（有无霉变）
- [ ] 运行测试（各程序正常）

**冰箱**:
- [ ] 清洁冷凝器（背面/底部散热器）
- [ ] 检查门封条（密封性）
- [ ] 清理排水孔
- [ ] 检查温度设置
- [ ] 清洁内部

**油烟机**:
- [ ] 清洗滤网
- [ ] 清洗风轮
- [ ] 清洁油杯
- [ ] 检查管道连接
- [ ] 测试各档位

## Installation Workflow

### State Flow
```
接单 → 上门勘测 → 确认条件 → 安装调试 → 客户验收 → 完成
```

### Installation Checklist (通用)

- [ ] 确认电源（电压、接地、线径）
- [ ] 确认水源（水压、管径）— 洗衣机/洗碗机
- [ ] 确认燃气（燃气类型、管路）— 燃气灶/热水器
- [ ] 确认排水位置
- [ ] 确认安装空间
- [ ] 安装设备
- [ ] 连接水电/气
- [ ] 运行测试
- [ ] 客户验收签字
